\itm Consider the two parametrizations of the line segment $S$ from $(0,0)$ to $(3,3)$ given by
\begin{align*}
\vec{x}_1(t) &= (t,t) \qquad 0 \leq t \leq 3 \\
\vec{x}_2(t) &= (2t,2t) \qquad 0 \leq t \leq 3/2. \\
\end{align*}
(a)  Evaluate the line integral $\int_S x^2\,dx + y\,dy$ using the parametrization $\vec{x}_1$. 

(b) Evaluate the line integral $\int_S x^2\,dx + y\,dy$ using the parametrization $\vec{x}_2$.

\sol{Using $\vec{x}_1$, we get 
\[
\int_{0}^3 t^2\, dt + t\,dt = \left[ t^3/3 + t^2/2 \right]_{0}^{3} = \boxed{27/2}
\]
Using $\vec{x}_2$, we get 
\[
\int_{0}^{3/2} (2t)^2\, 2dt + (2t)\,2dt = \left[ 4t^3/3 + t^2 \right]_{0}^{3/2} = \boxed{27/2}. 
\]
This result is consistent with the parametrization-independence of the line integral. 
}{\vfill}

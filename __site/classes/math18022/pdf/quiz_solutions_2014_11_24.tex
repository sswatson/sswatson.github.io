\itm Define $f:\R^3 \to \R$ by $f(x,y,z) = x^3y + \dfrac{xy^2}{1+z^2}$. 

(a) Find $\nabla \times (\nabla f)$. 

\iftoggle{solutions}{}{\vfill} 

(b) Calculate $\int_C \nabla f \cdot d\vec{s}$, where $C$ is the path contained in the $x$-$y$ plane shown below. 

\begin{center}
  \includegraphics{figures/path}
\end{center}

\iftoggle{solutions}{}{\vfill} 

\sol{(a) The answer is $\boxed{0}$, because conservative vector fields are curl-free. 

(b) The answer is $f((3,1,0))-f((-4,-1,0))=30-60=\boxed{-30}$, because $\int_{\vec{a}}^\vec{b} \nabla f \cdot d\vec{s} = f(\vec{b}) - f(\vec{a})$. 
}{}
